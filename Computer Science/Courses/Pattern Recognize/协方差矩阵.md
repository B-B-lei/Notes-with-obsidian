
#### 1.有关协方差矩阵的理解与相关实现:
$$C_{ij}=E{(x_i-m_i)(x_j-m_k)}$$上式为协方差的数学定义；协方差与方差计算方式类似，是用于衡量两个不同随机变量之间分布差异的数学量；这里的$C_{ij}$表示$X_i$与$X_j$两变量的协方差，而这两个变量分别为输入样本的第i个特征与第j个特征；组织成X_input matrix就是矩阵的第i和第j column;
其中的mi,mk分别代表xi与xk的均值，是两个标量；
显然对于任意两列特征都存在对应的Cov协方差；当特征维度N>=2时，可以通过协方差矩阵来表示一个Input矩阵X的协方差信息；

注意上述式子中的$x_{i/j}$表示的是标量元素，$C_{ij}$表示的是矩阵中索引为i,j的标量元素；等价的向量/矩阵语言表示协方差矩阵$C$为：
$$C=E\{(x-m)(x-m)^T\}$$
默认数学公式中的向量都是列向量，如$x^T$才是行向量；显然n x 1 向量与 1 x n 向量相乘得到n x n矩阵，这也称作两个向量的外积；矩阵乘法的一个理解角度为：AB = C  中C为所有A中列向量与B中对应行向量的外积的和矩阵；
上述式子中的E就是从外积的角度理解矩阵乘法的含义，$x$实际内容为每个样本；每个样本进行外积作为对最后协方差矩的贡献；
注：外积矩阵的秩为1；m个样本导致m个向量进行外积得到的矩阵进行求和后的矩阵秩<=m (每个外积矩阵最多贡献一个秩)。

在协方差矩阵中，每个元素$C_{ij}$如上式内容；假设原始输入为X，X内元素通过减去各自列所对应的mean得到$X_{mean}$,则协方差矩阵的计算过程可为：
$$C=X_{mean}^T\cdot X_{mean}$$
其中$X_{mean}$是$m\text{ x }n$矩阵,转置为$n\text{ x }m$矩阵， 故C结果为n x n矩阵；
具体实现过程明确后，实际操作可直接调用numpy中直接操作C=np.cov(X)得到X的Cov_matrix

上述两个矩阵乘法的差别在于：第一个式子严格按照数学定义，并且将样本当作默认的列向量看待，以矩阵外积和的角度理解矩阵乘法；
第二个式子以矩阵乘法的row column 细颗粒化看待矩阵乘法，并且将样本向量x看作行向量，$X^T$的步骤就是为了将行向量转为列向量；这样新的行向量就是所有相同特征索引下的特征向量；可以方便通过矩阵乘法传统定义得到具体的$C_{ij}$协方差元素。

#### 2.有关协方差矩阵的应用：
主要是计算后验概率需要联合概率，联合概率通过似然函数做跳板计算，而似然函数需要假设样本处于某种概率分布比如Gaussian分布，而高斯分布当涉及到多元特征样本时需要使用协方差矩阵计算；
