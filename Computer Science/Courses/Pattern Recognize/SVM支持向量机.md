## 1.函数间隔与几何间隔
### 1.1几何间隔的推导
**假设：**

- 超平面由方程 wTx+b=0 定义，其中 w 是法向量，b 是偏置项。
    
- 待求距离的点是 x1​。
    
- 超平面上任意一点是 x0​。
    

**推导步骤：**

1. **超平面上的点 x0​：** 由于 x0​ 在超平面上，所以它满足超平面方程： wTx0​+b=0⇒wTx0​=−b
    
2. **构建向量：** 从超平面上的点 x0​ 指向待求点 x1​ 的向量是 (x1​−x0​)。
    
3. **超平面的法向量方向：** 向量 w 垂直于超平面。为了得到单位法向量，我们需要将其归一化： n^=∣∣w∣∣w​
    
4. **投影：** 点 x1​ 到超平面的距离，就是向量 (x1​−x0​) 在超平面法向量方向上的投影的长度。 这个投影长度可以通过点积来计算： d=∣(x1​−x0​)⋅n^∣ d=∣(x1​−x0​)T∣∣w∣∣w​∣ d=∣∣w∣∣∣x1T​w−x0T​w∣​
    
5. **代入 x0​ 的性质：** 我们知道 x0T​w=wTx0​=−b。代入上式： d=∣∣w∣∣∣wTx1​−(−b)∣​ d=∣∣w∣∣∣wTx1​+b∣​
    

**这正是您推导出来，并且也是标准教科书上严谨的几何距离公式。**

### 接着联合 yj​

这个公式 d=∣∣w∣∣∣wTxj​+b∣​ 给出的是点 xj​ 到超平面的**正值距离**。

为了将类别标签 yj​∈{−1,1} 引入，我们考虑**函数间隔**： γ^​j​=yj​(wTxj​+b)

- 如果 yj​=1 且 xj​ 在正类一侧 (wTxj​+b>0)，那么 γ^​j​>0。
    
- 如果 yj​=−1 且 xj​ 在负类一侧 (wTxj​+b<0)，那么 yj​ 和 (wTxj​+b) 都是负数，所以乘积 γ^​j​>0。
    
- 如果分类错误，γ^​j​<0。
    

所以，我们可以将几何间隔 dj​ 写成**带符号的函数间隔**与 ∣∣w∣∣ 的比值： dj​=∣∣w∣∣yj​(wTxj​+b)​=yj​(∣∣w∣∣wTxj​+b​)

**这个公式 (也就是 PPT 截图的最后一行) 才真正是结合了类别信息，且符号与分类正确性一致的几何间隔，它在分类正确时为正，在分类错误时为负。** 这也是 SVM 中最终使用的几何间隔定义。


## 2 函数间隔与几何间隔的关系

## 2.1. **函数间隔 $\hat{\gamma}_i = y_i(w^T x_i + b)$ 的本质：**
    
    - 确实，它是一个确保值为正且不改变 (分类正确时) 的机制。
        
    - 它的作用就是将 $w^T x_i + b$ 的**符号**与真实标签 $y_i$ 的符号统一起来。
        
    - 如果分类正确，$\hat{\gamma}_i$ 是一个正数，其大小表示点到超平面的“置信度”或“远近”。
        
    - 如果分类错误，$\hat{\gamma}_i$ 是一个负数。
        
## 2.2 **几何间隔 $d_i = \frac{|w^T x_i + b|}{||w||}$：**
    
    - 这是点 $x_i$ 到超平面 $w^T x + b = 0$ 的**真实正值距离**。您自己推导的这个公式是完全严谨的。
        
    - 您的思考没错，分子需要有绝对值来保证距离是正的。
        
## 2.3 **函数间隔与几何间隔的关系：**
    
    - 它们之间的关系是 $d_i = \frac{\hat{\gamma}_i}{||w||}$ (当 $\hat{\gamma}_i > 0$ 时，或者更一般地，使用 $y_i(w^T x_i + b)$ 的形式来代替 $|w^T x_i + b|$ )。
        
    - 所以，函数间隔和几何间隔之间**只差一个 $||w||$ 作为分母**。
        
    - 您的结论是正确的：当 $||w||=1$ 时，几何间隔 $d_i$ 就等同于函数间隔 $\hat{\gamma}_i$。
        
## 2.4 **超平面缩放的不变性 (以及您纠正的理解)：**
    
    - **超平面本身：** $w^T x + b = 0$ 和 $(\lambda w)^T x + (\lambda b) = 0$ ($\lambda \neq 0$) 表示的是**完全相同的超平面**。它的空间位置和方向都没有变。
        
    - **函数间隔的变化：** 如果 $w, b$ 同时乘以一个正数 $\lambda$，那么函数间隔会变成 $\hat{\gamma}_i' = y_i(\lambda w^T x_i + \lambda b) = \lambda [y_i(w^T x_i + b)] = \lambda \hat{\gamma}_i$。**函数间隔会随缩放因子 $\lambda$ 同比例变化。**
        
    - 几何间隔的不变性：
        
        $d_i' = \frac{|(\lambda w)^T x_i + (\lambda b)|}{||\lambda w||} = \frac{|\lambda (w^T x_i + b)|}{|\lambda| ||w||} = \frac{|\lambda| |w^T x_i + b|}{|\lambda| ||w||} = \frac{|w^T x_i + b|}{||w||} = d_i$。
        
        几何间隔确实不会因为 $w, b$ 的同比例缩放而改变。它反映的是点到超平面的真实距离，而超平面并没有移动。

### 您的理解与 SVM 核心思想的完美契合

您刚刚进行的思考和纠正，正是 SVM 推导中最为精妙的部分：

1. **最大化几何间隔是目标**：我们希望找到一个超平面，使得距离它最近的样本点（支持向量）的**几何间隔**最大化。即 $\max d_{\text{min}}$。
    
2. 利用函数间隔简化优化：由于几何间隔对于 $w, b$ 的同比例缩放是不变的，而函数间隔是会同比例缩放的，这给了我们一个“自由度”。
    
    我们可以通过缩放 $w, b$（不改变超平面位置），使得所有支持向量的函数间隔恰好等于 1。
    
    此时，最小的函数间隔 $\hat{\gamma}_{\text{min}} = 1$。
    
3. 优化目标的转化：
    
    既然 $\hat{\gamma}_{\text{min}} = 1$，那么我们最大化最小几何间隔 $d_{\text{min}} = \frac{\hat{\gamma}_{\text{min}}}{||w||}$ 就等价于最大化 $\frac{1}{||w||}$。
    
    而最大化 $\frac{1}{||w||}$ 又等价于最小化 $||w||$，或者为了方便数学处理，最小化 $\frac{1}{2}||w||^2$。
    
4. 最终的优化问题：
    
    最小化 $\frac{1}{2}||w||^2$
    
    约束条件：对于所有样本点 $i$，有 $y_i(w^T x_i + b) \ge 1$。
    
    （这个约束条件确保了所有样本点都被正确分类，并且它们至少在函数间隔为 1 的边界之外。）
